香港中文大学 EES（地球与环境科学）课程亮点详细介绍

我已经阅读了你给的 CUHK EES 本科项目网页，并把网页里明确提到的所有课程名称/课程代码整理出来。需要说明：该页面只列出课程名（和少量课程代码），没有逐门课的官方 syllabus/大纲；下面每门课的“学什么”是根据课程名称、EES 常见课程体系与该网页对学科方向的描述做的合理推断式介绍（便于你快速理解课程用途）。课程清单主要来自网页“Highlights of EES Courses”区块及入学路径处列出的课程代码。( ees.cuhk.edu.hk )

一、网页中出现的课程（按网页分组）+ 中文内容介绍

1) Introductory Courses（入门课）

  • Exploring EES（探索地球与环境科学）：专业导论；了解地球圈层（岩石圈/水圈/大气圈/生物圈）如何相互作用，认识主要研究问题（气候变化、污染、自然灾害等）与学习路线。( ees.cuhk.edu.hk )
  • Solid Earth Dynamics（固体地球动力学）：板块构造、地幔对流、地震与火山成因、造山带与地壳演化等。( ees.cuhk.edu.hk )
  • Climate System Dynamics（气候系统动力学）：气候系统能量收支、大气/海洋环流、季风与年代际变化、气候反馈机制与变率。( ees.cuhk.edu.hk )
  • Geoscience Field Study（地球科学野外考察）：野外观察与记录、岩层与地貌识别、采样与测量、野外安全、现场数据整理（偏“体验+方法”）。( ees.cuhk.edu.hk )
  • Computational EES（地球与环境计算方法）：用编程/计算工具处理环境与地球数据（数据清洗、可视化、简单数值模拟或模型入门）。( ees.cuhk.edu.hk )
  • Environmental Science（环境科学）：环境过程基础、污染与生态影响、环境管理与可持续框架、案例分析。( ees.cuhk.edu.hk )
  • Environmental Chemistry（环境化学）：污染物的化学形态与迁移转化（大气/水/土），酸碱/氧化还原、营养盐循环、环境化学反应机制。( ees.cuhk.edu.hk )
  • Environmental Chemistry Lab（环境化学实验）：水/土/空气样品分析、滴定/光谱/色谱等常见分析方法、质量控制(QC/QA)、实验数据处理与报告写作。( ees.cuhk.edu.hk )
  • Environmental Engineering（环境工程）：水处理/废水处理、空气污染控制、固废处理、环境工程单元过程与工程设计概念。( ees.cuhk.edu.hk )

2)（网页继续列出的另一组“Introductory Courses”，实质更像中阶/方法基础课）

  • Analytical Methods for EES（地球与环境分析方法）：环境与地球样品的采集、前处理与仪器分析思路（如何选指标、如何做不确定性与误差分析）。( ees.cuhk.edu.hk )
  • Structural Geology（构造地质学）：褶皱、断层、应力应变、岩层变形历史解读；为地质工程与地震构造研究打基础。( ees.cuhk.edu.hk )
  • Geoscience Field Course（地球科学野外课程）：更系统的野外测绘与剖面记录、地层/构造识别、野外数据到解释报告的完整流程。( ees.cuhk.edu.hk )
  • Physics of the Earth（地球物理基础）：地球内部结构、重力/磁场、地震波、热流等地球物理观测与物理机制。( ees.cuhk.edu.hk )
  • Seismic Data Processing（地震数据处理）：地震波形数据的滤波、拾取、成像/反演基本思想；与勘探地球物理、地震学研究直接相关。( ees.cuhk.edu.hk )
  • Atmospheric Dynamics（大气动力学）：控制大气运动的基本方程、环流、天气系统形成机制；为气象与气候方向核心课。( ees.cuhk.edu.hk )
  • Atmospheric Chemistry（大气化学）：臭氧/NOx/VOCs 反应链、二次气溶胶形成、污染物来源与转化、监测与控制思路。( ees.cuhk.edu.hk )
  • Environmental Protection and Pollution Control（环境保护与污染控制）：政策法规框架、监测与标准、污染控制技术与管理实践（更偏应用与治理）。( ees.cuhk.edu.hk )
  • Oceanography（海洋学）：物理海洋（环流、波浪、潮汐）、海洋化学/生物过程概览、海气相互作用与海洋在气候中的作用。( ees.cuhk.edu.hk )
  • Hydrogeology（水文地质学）：地下水流动、含水层、污染物在地下水中的迁移、抽水试验与地下水资源管理。( ees.cuhk.edu.hk )
  • Biogeochemistry（生物地球化学）：碳氮磷硫等元素在生态系统与地球圈层间的循环，气候变化与生态反馈。( ees.cuhk.edu.hk )
  • Environmental Instrumentation Techniques（环境仪器与监测技术）：传感器与监测设备原理（气体/水质/颗粒物等）、校准、数据记录与误差控制。( ees.cuhk.edu.hk )
  • Environmental Instrumentation Techniques Lab（环境仪器实验）：上手使用仪器与传感器、现场布设与数据采集、数据质量控制与实验报告。( ees.cuhk.edu.hk )
  • Environmental and Biochemical Toxicology（环境与生化毒理学）：污染物如何影响生物（剂量-反应、暴露途径、毒性机制）、生态风险评估基础。( ees.cuhk.edu.hk )
  • Environmental and Biochemical Toxicology Lab（毒理学实验）：基础生物实验与毒性测试思路、数据统计与结论表达。( ees.cuhk.edu.hk )
  • Ecosystems and Climate（生态系统与气候）：生态系统过程与气候之间的相互影响（碳汇、蒸散、陆气耦合等）。( ees.cuhk.edu.hk )
  • Principles of Ecosystems and Climate（生态系统与气候原理）：上一门的更基础版本或理论导向版本（概念框架、关键过程与典型案例）。( ees.cuhk.edu.hk )
  • Global Environmental Change（全球环境变化）：全球变暖、海平面上升、极端事件、土地利用变化与污染等综合议题与证据链。( ees.cuhk.edu.hk )

3) Upper-Level Courses（高阶课）

地质/工程/地球物理方向：

  • Solid and Fluid Mechanics（固体与流体力学）：工程与地球系统中的基础力学；为岩土工程、地球物理与大气海洋动力学建模打底。( ees.cuhk.edu.hk )
  • Soil Mechanics and Its Applications（土力学及应用）：土体强度、渗流、沉降、边坡稳定等；对岩土/地基工程非常关键。( ees.cuhk.edu.hk )
  • Engineering Geology / Rock Mechanics and Its …（工程地质/岩石力学相关课，网页标题被截断）：岩体结构、破坏机制、工程稳定性分析；与隧道、边坡、地基项目相关。( ees.cuhk.edu.hk )
  • Applied Geophysics（应用地球物理）：重磁电、地震勘探、地质体成像与反演；用于资源勘探、工程勘察、地下结构探测。( ees.cuhk.edu.hk )
  • Petrology（岩石学）：火成岩/变质岩形成与演化、矿物组成与岩浆过程；理解地球内部过程。( ees.cuhk.edu.hk )
  • Geomorphology（地貌学）：侵蚀、沉积、地貌演化与过程地貌；与自然灾害、流域管理、海岸变化等相关。( ees.cuhk.edu.hk )
  • Marine Geology and Geophysics（海洋地质与地球物理）：海底地形、沉积、海底构造与地球物理探测；用于海上工程与海洋构造研究。( ees.cuhk.edu.hk )
  • Earthquake Source Physics（地震震源物理）：断层破裂过程、应力积累与释放、震源机制；对地震风险评估有用。( ees.cuhk.edu.hk )

大气/气候方向：

  • Land-Atmosphere Interactions and Boundary-Layer Meteorology（陆气相互作用与边界层气象）：地表-大气能量/水分交换、边界层结构、对天气与空气质量的影响。( ees.cuhk.edu.hk )
  • Tropical Meteorology（热带气象）：季风、对流、台风/热带气旋、热带波动；与香港区域气候高度相关。( ees.cuhk.edu.hk )
  • Physics and Chemistry of Aerosol（气溶胶物理与化学）：颗粒物生成、增长、光学性质、健康与气候效应。( ees.cuhk.edu.hk )
  • Air Pollution Science and Engineering（空气污染科学与工程）：来源解析、扩散与化学反应、监测与控制工程措施。( ees.cuhk.edu.hk )
  • Advanced Topics in Atmospheric…（大气高级专题，网页标题被截断）：通常为专题研讨/前沿课（如极端天气、化学传输模型、观测技术等）。( ees.cuhk.edu.hk )
  • Cloud Dynamics（云动力学）：云微物理、对流与降水过程、云与辐射/气候的关系。( ees.cuhk.edu.hk )

环境化学/工程/健康方向：

  • Advanced Environmental Chemistry（高级环境化学）：更深入的反应机制、环境介质过程、复杂污染体系与高级分析。( ees.cuhk.edu.hk )
  • Chemical Treatment Processes（化学处理过程）：水处理/废水处理中化学单元过程（混凝、氧化、消毒等）与工艺设计。( ees.cuhk.edu.hk )
  • Environmental Impact Assessment（环境影响评估）：EIA框架、影响识别与预测、缓解措施、报告与合规流程。( ees.cuhk.edu.hk )
  • Environmental Impact Assessment Lab（环评实验/实务）：更偏项目式：做评估案例、监测/建模练习、写报告与展示。( ees.cuhk.edu.hk )
  • Environmental Health（环境健康）：环境因素（空气、水、化学品、噪声等）与人类健康影响、风险评估与公共健康框架。( ees.cuhk.edu.hk )

数据/建模/遥感方法：

  • Statistical Methods and Data Analysis for EES（统计方法与数据分析）：回归、假设检验、时间序列、空间数据基础；做科研/咨询都很关键。( ees.cuhk.edu.hk )
  • Numerical Methods and Modeling for EES（数值方法与建模）：数值求解、模型构建、误差与稳定性；用于气候/大气/水文/地球物理模拟。( ees.cuhk.edu.hk )
  • Remote Sensing – Principles and Applications（遥感原理与应用）：卫星/航空遥感、光谱与辐射、影像处理、土地覆盖/污染/灾害监测应用。( ees.cuhk.edu.hk )
